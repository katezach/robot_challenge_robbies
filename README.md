# Robot Challenge Robbies

This REAME is divided into two parts to provide instruction on how to run the script for each task, and provide our results.

## Run Instructions
### Task 1
Get the distance of the robot from the object:
```bash
python3 task1.py 1
```
Go to the object:
```bash
python3 task1.py 2
```

### Task 2
```bash
python3 task2.py
```
### Task 3
```bash
python3 task3.py
```
